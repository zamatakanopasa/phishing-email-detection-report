# Phishing Email Detection & Awareness Report
##  Overview
This project analyzes phishing email samples, identifies key indicators, and provides awareness guidelines to prevent phishing attacks.

## 🛠 Tools Used
- Google Message Header Analyzer
- MXToolbox
- Public phishing datasets

## 🔍 Approach
- Collected phishing email samples
- Analyzed email headers and content
- Identified phishing indicators
- Classified risk levels
- Created prevention guidelines
